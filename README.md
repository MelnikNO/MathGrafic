# MathGrafic


## 📋 Описание проекта

Это продолжение первой практической работы. Базовая игра «Инопланетное вторжение» была дополнена новыми функциями, которые делают игровой процесс более увлекательным и технологичным.

## 🎮 Возможности игры

### 1. Управление кораблём
- Перемещение вправо/влево с помощью клавиш-стрелок (← →)
- Стрельба по пришельцам при нажатии **Пробел**

### 2. Звуковые эффекты 🎵
- Звук выстрела (`laser.wav`)
- Звук уничтожения пришельца (`explosion.wav`)
- Звук потери жизни (`hit.wav`)
- Звук перехода на новый уровень (`level_up.wav`)
- Звук завершения игры (`game_over.wav`)

### 3. Система уровней 📈
С каждым новым уровнем сложность увеличивается:
- **Скорость пришельцев** ↑ (×1.05)
- **Скорость корабля** ↑ (×1.05)
- **Скорость пуль** ↑ (×1.05)
- **Количество пришельцев** ↑ (+2 в ряду, макс. 20)
- **Скорость падения флота** ↑ (×1.05, макс. 20)
- **Очки за убийство** ↑ (×1.5)
- **Шанс выстрела пришельцев** ↑ (×1.2, макс. 5%)
- **Количество рядов** ↑ (каждые 2 уровня)

### 4. Сохранение и загрузка прогресса 💾
- **S** — сохранить игру (уровень, очки, жизни, рекорд)
- **L** — загрузить сохранённую игру
- Данные сохраняются в файл `savefile.pkl`

### 5. Бонусы и интерактивные объекты ✨
При уничтожении пришельцев с вероятностью **15%** выпадают бонусы:

| Бонус | Эффект | Длительность | Цвет |
|-------|--------|--------------|------|
| ❤️ **+1 жизнь** | Восстанавливает одну жизнь | Мгновенно | Зелёный |
| 🛡️ **Щит** | Временная неуязвимость | 5 секунд | Синий |
| ⚡ **Усиленный выстрел** | Тройной выстрел с повышенным уроном | 5 секунд | Золотой |

### 6. Визуальные эффекты 🎨
- Изменение цвета уровня в зависимости от сложности
- Информационное окно при переходе на новый уровень
- Отображение количества пришельцев на экране
- Анимация появления бонусов
- Визуальный эффект активного щита

### 7. Интерфейс
- **Счёт** (правая верхняя часть)
- **Рекорд** (центр верхней части)
- **Уровень** (под счётом, цвет зависит от сложности)
- **Оставшиеся корабли** (левая верхняя часть)
- **Количество пришельцев** (под кораблями)

## 🚀 Сборка исполняемого файла

### Установка PyInstaller
```bash
pip install pyinstaller
```

### Сборка для Windows
Создайте файл `build_windows.bat`:
```batch
@echo off
pyinstaller --onefile --add-data "images;images" --add-data "sounds;sounds" alien_invasion.py
```

### Сборка для macOS/Linux
Создайте файл `build.sh`:
```bash
#!/bin/bash
pyinstaller --onefile --add-data "images:images" --add-data "sounds:sounds" alien_invasion.py
```

После сборки исполняемый файл будет находиться в папке `dist/`.
